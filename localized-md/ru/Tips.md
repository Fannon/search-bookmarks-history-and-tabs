### ⌨️ Горячие клавиши

- **Открыть/закрыть расширение**: По умолчанию <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>.</kbd>. Вы можете изменить это сочетание в настройках расширений вашего браузера.
- **Переключить режим поиска**: Нажмите <kbd>Ctrl</kbd> + <kbd>F</kbd> для быстрого переключения между **Точным** и **Нечетким** поиском.
- **Навигация в стиле Vim / Emacs**:
  - <kbd>Ctrl</kbd> + <kbd>N</kbd> или <kbd>Ctrl</kbd> + <kbd>J</kbd> перемещает выделение **вниз**.
  - <kbd>Ctrl</kbd> + <kbd>P</kbd> or <kbd>Ctrl</kbd> + <kbd>K</kbd> перемещает выделение **вверх**.
- **Открыть результат**:
  - <kbd>Enter</kbd>: Открыть в новой активной вкладке (или переключиться на существующую вкладку).
  - <kbd>Shift</kbd> + <kbd>Enter</kbd> или <kbd>Alt</kbd> + <kbd>Enter</kbd>: Открыть в **текущей вкладке**.
  - <kbd>Ctrl</kbd> + <kbd>Enter</kbd>: Открыть в фоновом режиме, не закрывая окно поиска.
  - <kbd>F2</kbd>: Редактировать выбранную закладку или создать новую из выбранного URL-адреса.
  - Совет: Включите опцию `openInCurrentTab`, чтобы инвертировать это поведение — тогда <kbd>Enter</kbd> будет открывать в **текущей вкладке**, а <kbd>Shift</kbd>/<kbd>Alt</kbd> + <kbd>Enter</kbd> — в **новой**.
- **Скопировать URL**: Щелкните **правой кнопкой мыши** по любому результату или используйте настроенное сочетание клавиш.
- **Гибридный поиск**: Нажмите <kbd>TAB</kbd>, чтобы быстро вставить двойной пробел-разделитель для объединения фильтров таксономии и поискового запроса (например, `#работа  проект`).

### 🔍 Расширенный поиск

- **Режимы поиска**: Используйте специальные префиксы для фильтрации по типу:
  - `#тег`: Только закладки с этим тегом.
  - `~папка`: Только закладки в этой папке.
  - `t `: Только открытые вкладки.
  - `b `: Только закладки.
  - `h `: Только история и вкладки.
  - `s `: Только поисковые системы.
- **Фильтрация И (AND)**: Объединяйте маркеры для точных результатов. Например, `#работа #todo` найдет закладки, содержащие оба тега, а `~Проекты #дизайн` найдет тег дизайн внутри папки Проекты.
- **Интерактивные значки**: Нажмите на любой значок **тега** или **папки** в результатах поиска, чтобы мгновенно отфильтровать список по этому элементу.
- **Прямой переход**: Введите домен, например `example.com` или `localhost:3000`, чтобы сразу перейти на этот адрес.
- **Быстрые алиасы**: Используйте `g <запрос>` для Google, `d <запрос>` для dict.cc или настройте собственные алиасы в параметре `customSearchEngines`.

### 🚀 Настройка производительности

Если открытие расширения или поиск кажутся медленными:

- **Переключитесь на Точный поиск**: Точный поиск (Precise) может работать быстрее, чем Нечеткий (Fuzzy), на очень больших коллекциях закладок.
- **Ограничьте результаты**: Уменьшите параметр `searchMaxResults` (по умолчанию: 24) для ускорения отрисовки результатов.
- **Оптимизируйте историю**:
  - Уменьшите значения параметров `historyMaxItems` (по умолчанию: 1024) или `historyDaysAgo` (по умолчанию: 14).
  - Используйте `historyIgnoreList` для пропуска частых адресов (например, `localhost`, `google.com`).
  - Периодически очищайте историю браузера; переполненная база данных истории — главная причина медленного запуска расширения.
- **Отключите значки**: Установка параметров `displayTags: false` или `displayFolderName: false` скрывает соответствующие значки на панели результатов, но сохраняет работоспособность поиска по ним, существенно экономя ресурсы процессора.

### 🛠️ Советы по организации

- **Избранное**: Используйте кнопку FAVORITE (звездочка) в редакторе закладок для переключения приоритетов: без избранного, желтая звезда (`+25`), оранжевая звезда (`+50`) и красная звезда (`+75`).
- **Собственные баллы релевантности**: Вы можете повысить приоритет важных закладок, добавив ` +20` (или любое другое число) к названию закладки.
  - *Пример*: `Панель мониторинга +50 #работа`
- **Исключаемые папки**: Используйте параметр `bookmarksIgnoreFolderList` для полного сокрытия архивных или конфиденциальных папок из результатов поиска.
- **Индикаторы открытых вкладок**: Закладка, которая уже открыта в одной из вкладок, получает бонус к оценке релевантности и специальную градиентную окраску левой полосы (от цвета закладки к цвету вкладки).
